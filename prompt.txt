this here is a docker compose stack generator
compose.settings/ contains templates for general settings for configuring the stack
compose.templates/ contains templates for individual services
configs.templates/ contains templates for configuration files that go inside containers
containers.src/ contains dockerfile source for building custom containers
kotlin.src/ contains kotlin source for building custom services
build-datamancy-v3.kts is the build script
outputs in dist/
dist/docker-compose.yml is the final docker compose file
